# 错误码文档

## 成功状态

| 错误码 | 描述 |
|--------|------|
| 10000 | 业务处理成功 |

## 参数错误 (20000+)

| 错误码 | 描述 |
|--------|------|
| 20000 | 提供参数有问题 |
| 20001 | 参数缺失 |

## 鉴权错误 (30000+)

| 错误码 | 描述 |
|--------|------|
| 30000 | 鉴权失败 |
| 30001 | 访问令牌过期 |
| 30002 | 刷新令牌过期 |
| 30003 | 令牌等级不够，如stu无法进行审核 |
| 30004 | 没有 token |
| 30005 | 用户登出令牌已拉黑 |

## 业务错误 (40000+)

### 用户相关

| 错误码 | 描述 |
|--------|------|
| 40000 | 用户已存在 |
| 40001 | 邮箱格式不正确 |
| 40002 | 用户未激活、没有绑定相应邮箱 |
| 40003 | 密码错误 |
| 40004 | 用户不存在 |
| 40005 | 邮箱验证码已过期 |
| 40006 | 邮箱验证码不匹配 |
| 40007 | 等待两分钟重发邮件 |

### 赛事相关

| 错误码 | 描述 |
|--------|------|
| 40008 | 该赛事材料不存在 |
| 40009 | 判断上传的图片不是奖状或者荣誉证书 |
| 40010 | 表示未经过申诉，无法直接修改该材料 |
| 40011 | 上传的材料没被认定 |

### 记录相关

| 错误码 | 描述 |
|--------|------|
| 40012 | 该记录不存在 |

### 申诉相关

| 错误码 | 描述 |
|--------|------|
| 40013 | 申诉记录不存在 |
| 40014 | 申诉已存在 |
| 40015 | 用户对不属于自己的材料进行申诉、对不属于自己的申诉记录进行撤销、查看不属于自己的申诉 |
| 40016 | 用户尝试对已经处理的申诉进行撤销 |

### 维护相关

| 错误码 | 描述 |
|--------|------|
| 40017 | 学院不存在 |
| 40018 | 学院已存在 |
| 40019 | 专业已存在 |

## 服务错误 (50000+)

| 错误码 | 描述 |
|--------|------|
| 50000 | 内部服务错误 |
| 50001 | 文件处理错误 |
| 50002 | 数据库错误 |
| 50003 | Redis错误 |
| 50004 | 配置错误 |

## 使用说明

1. **错误码范围划分**：
    - 10000: 成功状态
    - 20000-29999: 参数错误
    - 30000-39999: 鉴权错误
    - 40000-49999: 业务错误
    - 50000-59999: 服务错误

2. **前端处理建议**：
    - 根据错误码进行业务逻辑处理
    - 对于通用错误类型，可统一处理
